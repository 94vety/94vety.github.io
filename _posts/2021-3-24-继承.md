# ES5 和 ES6 继承的区别

**ES6** 中类 **class** 的概念，类 **class** 的继承是通过 **extends** 来实现的，**ES5** 中是通过设置构造函数的 **prototype** 属性，来实现继承的。

***

# ES5 继承

```bash
function A() {
    this.a = 'hello'
}

function B() {
    A.call(this)
    this.b = 'world'
}

B.protptype = Object.create(A.prototype, {
    constructor: {
        value: B,
        writable: true,
        configurable: true
    }
});

let b = new B()
```

1. 构造函数 B 继承构造函数 A，首先让构造函数 B 的 prototype 对象中的 _proto_ 属性
