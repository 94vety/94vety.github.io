# 区域

1. ```Workspace``` **工作区**，就是我们最开始写代码的区域.<br>
2. ```Index``` **暂存区**，有时候也叫 stage，可以通过 ```git add xx文件``` 把想要提交的东西放到暂存区.<br>
3. ```Repository``` **本地仓库**，把暂存区的代码 ```commit``` 之后就放到了本地仓库，此时会有版本号.<br>
4. ```Remote``` **远程仓库**，可以 ```clone``` 到本地，本地也可以 ```push``` 上去.<br>
