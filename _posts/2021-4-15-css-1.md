# CSS3 新特性

CSS3 新增特性
    - CSS3 实现圆角（border-radius）
    - 阴影（box-shadow）
    - 对文字加特效（text-shadow）
    - 线性渐变（gradient）
    - 旋转（transform: rotate(9deg)）
    - 缩放（transform: scale(0.85, 0.90)）
    - 倾斜（transform: skew(-9deg, 0deg)）
    - 增加了更多的 CSS 选择器
