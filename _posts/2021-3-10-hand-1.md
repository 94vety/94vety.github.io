## 防抖与节流的区别

1. **防抖：**防抖是将多次执行变为最后一次执行。<br>
2. **节流：**节流是将多次执行变为每隔一段时间执行一次。<br>

***

## 防抖

#### 原理

**事件回调函数**在一段时间后才执行，如果在这段时间内再次调用则重新从 0 开始计算到 interval 的时间，在预定的时间内没有再次调用函数，则执行**事件回调函数**。

```bash
function debounce (fn, interval = 500) {
    let timer = null;

    return function() {
        clearTimeout(timer);

        timer = setTimeout(() => {
            fn(...arguments)
        }, interval)
    }
}
```
